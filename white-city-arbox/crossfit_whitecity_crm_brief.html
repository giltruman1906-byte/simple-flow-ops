CrossFit White City — Data pipeline MVP

End-to-end member analytics: extract gym data from Arbox API, model with medallion architecture, serve KPI dashboard to gym owner.

Client

CrossFit White City

Data source

Arbox API

MVP effort

~33 hours

Data scope

Last 3 months

Members Subscriptions Attendance Bronze / Silver / Gold dbt Core Streamlit DuckDB

Two-phase delivery

Phase 1 — MVP (now)
Storage DuckDB (local file)
Ingestion Python + requests
Transformation dbt Core + dbt-duckdb
Dashboard Streamlit (local)
Orchestration Manual / make all
Phase 2 — Live (post-approval)
Storage Supabase (PostgreSQL)
Ingestion Python + requests
Transformation dbt Core + dbt-postgres
Dashboard Metabase (public URL)
Orchestration Modal (daily cron)
